# Contexto de curso

Este set de laboratorios se puede usar como parte de un curso o diplomado de Analytics e Ingenieria de Datos.

Si usted lo usara como material asociado a "Analytics Edge (edX)", considere:

- Ajustar los prerequisitos de AWS segun el nivel (cuentas sandbox, budgets, SCPs)
- Reemplazar el dataset sintetico por un dataset institucional o un dataset publico permitido
- Agregar rubricas de evaluacion (tiempo, costo, correctitud, observabilidad)

Sugerencia de secuencia didactica:

- Lab 00: Bootstrap
- Lab 01: Glue ETL basico
- Lab 02: Asistencia con prompts (estandar de codigo)
- Lab 03: Orquestacion y consultas
- Lab 04: Agente de operaciones (acciones verificables)
- Lab 05: Troubleshooting
- Lab 06: Calidad de datos
- Lab 07: Integracion con n8n
