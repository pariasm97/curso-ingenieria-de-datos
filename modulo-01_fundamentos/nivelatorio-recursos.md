# Ruta de Nivelación en SQL y Python — 
---

## Nivel 1 — Fundamentos prácticos

Aprender la sintaxis esencial de Python y SQL y resolver tareas reales de manipulación y consulta de datos.

### Python Básico

**Competencias:**

- Sintaxis, estructuras de control y funciones
- Tipos de datos, listas, diccionarios
- Lectura y escritura de archivos
- Introducción a `pandas` y `numpy`


**Recursos con práctica:**
- [Python para Principiantes — Microsoft Learn](https://learn.microsoft.com/es-es/training/paths/beginner-python/)  
- [Curso completo de Python — freeCodeCamp (YouTube, 4h)](https://www.youtube.com/watch?v=rfscVS0vtbw)  
- [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/)  

---

### SQL Básico

**Competencias:**
- `SELECT`, `WHERE`, `ORDER BY`, `GROUP BY`, `HAVING`
- Funciones agregadas
- Conceptos de claves y relaciones

**Recursos con práctica:**
- [SQLBolt](https://sqlbolt.com/) — *Lecciones interactivas paso a paso.*
- [Mode Analytics SQL Tutorial](https://mode.com/sql-tutorial/) — *Consultas reales con resultados en vivo.*
- [Khan Academy — Introducción a SQL](https://es.khanacademy.org/computing/computer-programming/sql) — *Ejercicios autocorregibles con retroalimentación.*

---

## Nivel 2 — Análisis y manipulación de datos

### Objetivo:
Aprender a transformar, limpiar y analizar datos de fuentes reales, conectando Python y SQL.

### Python 

**Competencias:**
- Limpieza y exploración de datos con `pandas`
- Lectura de APIs y archivos JSON/CSV
- Uso de entornos virtuales y `requirements.txt`

**Recursos con práctica:**
- [Data Analysis with Python — freeCodeCamp (YouTube, 10h)](https://www.youtube.com/watch?v=r-uOLxNrNk8)  
- [Kaggle Learn: Data Cleaning](https://www.kaggle.com/learn/data-cleaning)  
- [Kaggle Learn: Pandas](https://www.kaggle.com/learn/pandas)  

---

### SQL 
**Competencias:**
- `JOIN`, `UNION`, `CASE`
- Subconsultas y CTEs (`WITH`)
- Funciones de ventana (`OVER`, `PARTITION BY`)
- Optimización básica de consultas

**Recursos con práctica:**
- [Mode Analytics — SQL avanzado y Window Functions](https://mode.com/sql-tutorial/sql-window-functions/)  
- [LeetCode SQL](https://leetcode.com/problemset/database/)  
- [Hackerrank SQL Track](https://www.hackerrank.com/domains/sql)

## Nivel 3 — Ingeniería y optimización

**Competencias:**
